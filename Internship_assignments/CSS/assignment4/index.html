Box sizing

1. Block Element:

A block-level element always starts on a new line, and the browsers automatically add some space before and after the element. A block-level element always takes up the full width available.

Some block level elements are:

  • H1-H6
  • dt
  • Header
  • video
  • Nav
  • ol
  • article
  • footer
  • figure
  • figcaption
  • address

2. Inline Element:

An Inline element does not start on a new line. An inline element only takes up as much width as necessary.

Some inline level elements are:

  • strong
  • button
  • input
  • label
  • sub
  • select
  • map
  • a
  • sup
  • var
